# python-learning-manager

A single-file HTML web app to track Python learning progress with AI prompts, study logs, and custom roadmaps.

## Features
- 📊 **Dashboard**: Track overall progress, study streaks, and weekly hours.
- 🗺️ **Custom Roadmap**: Edit, add, or remove phases and modules.
- 🤖 **AI Prompts**: One-click copy for LEARN, PRACTICE, TEST, and PROJECT prompts.
- 📂 **Project Tracker**: Track your GitHub projects for portfolio building.
- 🌙 **Dark Mode**: Eye-friendly dark theme for night study sessions.
- 📥 **Import/Export**: Backup or import your learning data via JSON files.

## How to use
1. Download or clone this repository.
2. Open `python_manager.html` in your web browser.
3. Set up your profile and start tracking!
